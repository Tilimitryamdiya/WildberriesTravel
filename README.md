## Тестовое задание от компании Wildberries

### Первый экран ###

<img src="pic/first_screen.PNG" width="25%" height="25%">

Содержит список актуальных дешевых авиаперелетов:
- Город отправления.
- Город прибытия.
- Дата отправления.
- Дата возвращения.
- Цена в рублях.
- Иконка “Лайк” (имеет два состояния).
